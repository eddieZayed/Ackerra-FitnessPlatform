# Ackerra - AI-Enhanced Fitness and Wellness Platform

**Ackerra** is an all-in-one fitness and wellness platform developed as my graduation project at Birzeit University.

It offers a complete set of tools to support users on their health journey, including:
- Personalized meal analysis through NutriSearch
- Customized workout plans and behavior change tools
- Emotional support chatbot powered by AI
- A shop for fitness supplements, clothing, and equipment

Built using modern technologies like React, MUI, Node.js, MongoDB, and TypeScript, Ackerra combines fitness coaching, nutrition analysis, and shopping in a seamless, user-friendly experience.

## Technologies Used
- Front-end: React, TypeScript, Material-UI (MUI), Vite
- Back-end: Node.js, Express.js
- Database: MongoDB
- APIs: Edamam API, ExerciseDB API, OpenAI API
- Libraries: Formik, Yup, Axios, React Router, Chart.js, Framer Motion

## Key Features
- NutriSearch tool for detailed nutrition analysis
- Fitness calculators: BMI, Calorie, Body Composition, Calories Burned
- Personalized training programs generator
- Emotional support chatbot (Ackerra)
- Behavior change assessment tool
- Fitness-related shop with seller dashboard
- Responsive and mobile-friendly design

## Project Goal
To empower users by providing affordable, smart, and personalized fitness coaching and wellness tools — all in one platform.
